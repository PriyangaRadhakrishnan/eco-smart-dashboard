# EcoSmart AI Dashboard

A premium, glassmorphic energy and carbon monitor that helps you track your electricity usage and climate impact in real-time.

## 🚀 Live Demo
[https://priyangaradhakrishnan.github.io/eco-smart-dashboard/](https://priyangaradhakrishnan.github.io/eco-smart-dashboard/)

## ✨ Features
- **Real-time Carbon Tracking**: Monitor CO2 emissions based on device usage.
- **Energy Waste Detection**: AI-powered alerts for empty rooms or daylight waste.
- **Financial Analytics**: Live conversion of carbon footprint to kWh and cost (₹).
- **One-Click Optimization**: Instantly reduce waste with smart automation.

## 🛠️ Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
